Open PowerShell as Administrator:

Navigate to the script directory (path depends on where you placed it): 
            cd D:\SysConfig\ABP_UWP_App_Remover

Dry-run mode:
.\ABP_UWPAppRemover_EN.ps1 -DryRun

Execute after confirming the audit list:
.\ABP_UWPAppRemover_EN.ps1

Key advantages: The script uses the -AllUsers parameter and wildcard matching to handle cases that Geek Uninstaller may miss:

1.Multi-user installation instances

2.System-level provisioned packages

3.Residuals from different app versions
